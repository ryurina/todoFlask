# todoFlask
Recently, I learned Flask, so I decided to make todo-app as practicing my knowledge
-------------------------
## Requirement:
- Python 3.6 or later
- Flask 

--------------------------
## Run the app:
- Open Terminal/CMD
- Type these commands
Linux:
```
sudo python3 todoapp.py
```
Windows:
```
python todoapp.py
```
- Open browser and type the url " http://127.0.0.1:5000/ "
